Bezier Flower

Bezier Flower Animation.

Example code showing how to group Bezier curves to make complex shapes

Command Keys:

  • 'A' : Toggle Animation
  • 'B' : Toggle Blur motion
  • 'C' : Toggle Bezier control lines.
  • 'H' : Toggle Help 'Hud'
  • 'I' : Toggle Colour Interpolation.
  • 'M' : Toggle Flower movement
  • 'N' : Toggle stroke.
  • 'O' : Toggle Noise (CPU intensive).
  • 'S' : Toggle Stem flowers
  • '+' : Increase number of petals.
  • '-' : Decrease number of petals.
  • [RIGHT] : Increase length of petals.
  • [LEFT] : Decrease length of petals.
  • [UP] : Increase tilt of petals.
  • [DOWN] : Decrease tilt of petals.
  • 'R' - Reset to default parameters.
Make sure you select and 'focus' the animation window before applying any commands.

Based on my own code from the Open University 'Mass Collaboration' book on Computer Art using Java Processing (2013 and 2019). A book I worked on with over 40 other students and faculty members from the Maths and Computer department.
