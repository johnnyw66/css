Retro Game Demo

PLEASE WAIT FOR DOWNLOAD TO COMPLETE!

Retro game demo - written in Javascript - ported from a portion of game code written in Lua. The game was released on the Playstation 3 in 2011 for Sony's Home application.

After working on the original game application - I wrote this demo to give me some practice coding in Javascript. The game is in the style of arcade games from the 1990s - and so some of the game play was pretty naive. The turning mechanism on the terrain vehicles is embarrassing! :(

Although I didn't port any game play code - the one useful by-product which came out of this exercise, was that I had a pretty cool tile map demo. I envisage the Tilemap routine - (which is about a page of code) could be used in some business application.
