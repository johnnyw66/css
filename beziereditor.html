Bezier Editor

Simple Bezier curve editor with animation.

Shows how the Bezier point is calculated on a parametric curve by displaying the recursive line segments which make up each point on that curve.

Using your mouse - click on either the end points (drawn as squares) or the control points (drawn as circles) and drag to its new position. Releasing the mouse will set the new position for the selected point.

Command Keys:

  • 'A' : Toggle Animation
  • 'P' : Pause/Restart Animation
  • '+' : Increase Animation speed.
  • '-' : Decrease Animation speed.
  • 'R' : Reset the Bezier curve to its default definition
Make sure you select and 'focus' the animation window before applying any commands.
